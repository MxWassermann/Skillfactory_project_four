# Учебный проект от Skillfactory №4: "Предсказание склонности клиента банка к открытию депозита"
## Суть проекта
Имеются данные о последней маркетинговой кампании, которую проводил банк: задачей было привлечь клиентов для открытия депозита. 
Необходимо проанализировать эти данные, выявить закономерность и найти решающие факторы, повлиявшие на то, что клиент вложил деньги именно в этот банк. 
Успех поднимет доходы банка и поможете понять целевую аудиторию, которую необходимо привлекать путём рекламы и различных предложений.
### Задача
**Бизнес-задача**: определить характеристики, по которым можно выявить клиентов, более склонных к открытию депозита в банке, и за счёт этого повысить результативность маркетинговой кампании.

**Техническая задача**: построить модель машинного обучения, которая на основе предложенных характеристик клиента будет предсказывать, воспользуется он предложением об открытии депозита или нет.
## Структура проекта
1. Первичная обработка данных
2. Разведывательный анализ данных (EDA)
3. Отбор и преобразование признаков
4. Решение задачи классификации: логистическая регрессия и решающие деревья
5. Решение задачи классификации: ансамбли моделей и построение прогноза
